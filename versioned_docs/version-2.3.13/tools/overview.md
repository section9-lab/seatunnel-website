# SeaTunnel Ecosystem Overview

Apache SeaTunnel Ecosystem is a collection of ecosystem projects focused on developer and operator productivity, covering LLM integration, configuration conversion, and AI-powered assistance.

## Ecosystem Projects

| Project | Purpose | Status |
|---------|---------|--------|
| [SeaTunnel Skill](seatunnel-skill) | Claude AI integration for SeaTunnel operations | Available |
| [SeaTunnel MCP Server](seatunnel-mcp) | Model Context Protocol server for LLM integration | Available |
| [x2seatunnel](x2seatunnel) | Configuration converter (DataX → SeaTunnel) | Available |

## Source Repository

All ecosystem projects are maintained in the [SeaTunnel Tools](https://github.com/apache/seatunnel-tools) repository.
